REGRAS

  • Uma palavra aleatória (e pouco comum) do dicionário será sorteada.
  • Cada jogador deve criar uma definição para essa palavra, seja inventada para enganar ou tentando se aproximar da definição real.
  • Quando todos terminarem, todas as definições serão exibidas juntas. Inclusive a original.
  • O jogador tenta descobrir qual delas é a verdadeira. Se acertar, ganha 1 ponto.
  • Se um jogador escolher uma definição falsa criada por outro participante, quem a escreveu ganha 1 ponto por enganar.
